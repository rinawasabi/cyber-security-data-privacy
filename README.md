# Cybersecurity and Data Privacy

This repository is for the course **Cybersecurity and Data Privacy**.

## Booking System Project

The Booking System project was completed as part of the course. The work included setting up the Booking System with Docker, performing penetration testing, documenting findings, and improving parts of the system.

The testing covered topics such as access control, authentication, password security, authorization, and GDPR-related requirements.

I also practiced web application security through PortSwigger labs, including SQL injection, path traversal, OS command injection, SSRF, XSS, and information disclosure.
