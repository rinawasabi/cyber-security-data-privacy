## 4. Data Retention
Personal data is stored only for as long as necessary to provide the service.  
Inactive or deleted accounts are securely erased from our database within **90 days** of deletion.  
Logs related to system operation are retained for up to **6 months** for security and audit purposes.

## 5. Data Security
We implement appropriate **technical and organizational measures** to protect personal data, including:
- Password hashing using bcrypt
- Role-based access control (reserver vs. administrator)
- Limited access to data by authorized personnel only
- Regular audits of access logs and permissions
